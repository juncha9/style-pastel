Pastel highlights inline styles

Each declaration is tinted by category — layout, box, typography, surface, motion, and other. The property name carries the color; the value is left untouched.

Custom property anchor and value references should both pick up the slate tint.
rgba inside value (with commas and semicolons) must not split the declaration: rgb(0, 0, 0); calc(100% - 16px)
